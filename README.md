Expériences LED / Laser - Notebook Python
=========================================


Installation rapide (Python pur)
----------------------------------
1. Cloner le dépôt ou télécharger les fichiers
```bash
git clone https://github.com/TortueSandwich/tp_optiques____.git
cd tp_optiques____
```

2. Créer un environnement Python isolé
```bash
python3 -m venv env

# (Linux/macOS)
source env/bin/activate
# (Windows)
env\Scripts\activate   

pip install notebook pandas matplotlib numpy

jupyter notebook
```
